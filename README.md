# 📰 News Website

A simple and responsive **News Web App** that fetches real-time news using the **GNews API**.  
Users can browse trending topics, search news, and view articles by category like Technology, India, Politics, etc.

---

## 🚀 Live Demo
👉 https://kathier02.github.io/NewsSite/

---

## 📸 Features

- 📰 Real-time news fetching using API
- 🔎 Search functionality for custom news
- 📂 Category-based news (Technology, India, Politics, etc.)
- 📱 Responsive design (mobile + desktop)
- 🖼️ Dynamic news cards with images and source info
- ⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GNews API

## 🔧 How It Works

1. User selects a category or searches a topic
2. JavaScript sends request to GNews API
3. API returns latest articles
4. UI dynamically renders news cards

---

## ⚠️ Known Issue

- API requests may fail on GitHub Pages due to **CORS policy restrictions**
- Works perfectly on local development (VS Code)
